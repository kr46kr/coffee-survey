# NovaBrew Coffee Taste Profile Quiz

A personality-based coffee matching quiz built for NovaBrew Coffee Co. The experience helps subscribers discover their primary and secondary coffee identity, then connects that result to recommended NovaBrew coffees.

## Why This Exists

NovaBrew's retention challenge is not poor coffee quality. It is a lack of personalization. This prototype turns the strategy recommendation from Module 1 into a working product concept: a premium quiz that makes subscribers feel known from day one.

## Coffee Personalities

- **Bold Explorer**: intense, high-character coffee with strong energy and presence.
- **Smooth Operator**: balanced, polished, easy-to-love coffees that feel refined and dependable.
- **Cozy Classic**: warm, familiar, ritual-friendly coffees built for comfort and routine.
- **Wild Card**: experimental, surprising coffees for people who want novelty and discovery.
- **Undefined Palate**: guided exploration for subscribers who are still figuring out what they like.

## Features

- 6-question personality quiz
- Primary + secondary result logic
- Coffee recommendations tied to each result
- Premium minimal visual design with stronger headline typography
- Result imagery for each personality type
- Mobile-responsive quiz flow

## Built With

- Next.js
- React
- Tailwind CSS

## About

Built as a customer-personalization prototype to improve subscriber retention through taste-profile matching and a more identity-rich subscription experience.
